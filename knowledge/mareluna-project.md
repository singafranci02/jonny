# Project Mareluna

Mareluna is my side project: a tide-prediction app for free divers on the
Ligurian coast. The MVP deadline is September 20th, 2026.

Tech decisions so far:
- Backend in Rust (axum), hosting on Fly.io
- Tide data from the EMODnet bathymetry API
- The app name comes from "mare" + "luna" because tides follow the moon

Open question: whether to launch iOS-first or as a PWA.
